DjangoPyPI 2

DjangoPyPi2 is a fork of the original DjangoPyPi project, providing re-implementation of the Python Package Index <http://pypi.python.org>, with awesome user-interfaces based on Twitter Bootstrap and a full web management interface.

Installation

$ virtualenv pypi-site
$ source pypi-site/bin/activate
$ pip install django south djangopypi2

Configuration

In your settings.py

INSTALLED_APPS = (
    ...
    'south',
    'djangopypi2.apps.frontend',
    'djangopypi2.apps.bootstrap',
)

TEMPLATE_CONTEXT_PROCESSORS = (
    'django.contrib.auth.context_processors.auth',
    'django.core.context_processors.request',
)

STATIC_ROOT = '/path/to/static/root'
MEDIA_ROOT  = '/path/to/media/root'

In urls.py

urlpatterns = patterns("",
    ...
    url(r'', include("djangopypi2.apps.frontend.urls"))
)

Seal everything up

$ ./manage.py syncdb
$ ./manage.py migrate djangopypi2.apps.frontend
$ ./manage.py loadclassifiers
$ ./manage.py collectstatic
